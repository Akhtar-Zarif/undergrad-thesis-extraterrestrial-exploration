# Scientific Exploration of Extraterrestrial Planets using Quantum Computing

## Undergraduate Thesis Project

### Overview
This repository contains the undergraduate thesis and project documentation for a simplified Mars Exploration Rover model designed to serve as the 'eyes and brain' of a planetary scientist.

### Project Details
- **Institution**: American International University - Bangladesh (AIUB)
- **Faculty**: Faculty of Engineering
- **Department**: Computer Engineering
- **Supervisor**: Dr. M. Tanseer Ali
- **Semester**: Fall 2019-2020
- **Date**: December, 2020

### Key Features
- 🚀 **Computer Vision System**: Rock detection and segmentation using machine learning (88.8% accuracy)
- 🔬 **Spectrophotometer**: LIBS-based instrument for elemental composition analysis
- 💻 **Quantum Computing**: Experimental analysis using IBM quantum computers
- 🤖 **Prototype Rover**: Custom-designed rover with integrated sensors

### Technologies Used
- TensorFlow
- OpenCV
- YOLO
- Raspberry Pi 4
- Coral USB Accelerator
- IBM Quantum Experience

### Results
- **Image Processing**: 88.8% accuracy in rock detection and segmentation
- **Spectrometer**: 60-65% accuracy in elemental identification
- **Quantum Computing**: Successful demonstration of 'true' randomness using qubits
